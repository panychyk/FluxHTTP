# FluxHTTP

FluxHTTP is a lightweight, composable HTTP networking framework for Swift built around a decorator-based pipeline architecture.

It allows you to extend networking behavior (retry, caching, timeout, logging, etc.) without modifying core networking logic.

---

## ✨ Features

- Async/await native
- Decorator-based architecture
- Fully composable HTTP pipeline
- Lightweight and extensible design
- Easy to mock and test
